Etapa 1: Ingeniería de Modelo

Predicción de Diabetes en Mujeres Embarazadas

Se seleccionó un algoritmo de clasificación, como Random Forest o Regresión Logística, adecuados para predecir un resultado binario como “diabetes: sí o no”.

El modelo se entrenó con el conjunto de datos Pima Indians Diabetes Dataset, usando variables como glucosa, presión arterial, edad, entre otras.
