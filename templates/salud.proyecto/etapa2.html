Etapa 2: Ingeniería de los Datos

Predicción de Diabetes en Mujeres Embarazadas

  • Selección: Se eligen las variables más relevantes para el problema.
  • Limpieza: Se corrigen errores, se eliminan duplicados y se gestionan los valores faltantes.
  • Transformación: Se convierten los datos a formatos adecuados (por ejemplo, normalización).
  • Documentación: Se registra cada paso para asegurar trazabilidad y reproducibilidad.

Preparación final: Con todo lo anterior, los datos están listos para el modelado. Se crean nuevas características (feature engineering) y se normalizan para mejorar el rendimiento del modelo.
